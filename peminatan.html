    Pemrograman

    The essence of technology development. Programming is the process of writing, testing, and maintaining code to make an application, software, or system work.

    UI/UX

    This field focuses on how applications look (UI) and how users feel when using them (UX). The goal is to create an interface that is attractive, easy to use, and enjoyable for the user.

    Network

    This field studies how devices connect and exchange data, both locally and over the internet. The main focus is on the security, speed, and stability of data communications.

    Data

    Processing data into useful information. This field includes the collection, analysis and visualisation of data to support better, evidence-based decision-making.
